We have used cloudflare SSL but this domain is still accessible as http://mern-stack.com. Lets force http to https by creating a page rule in cloudflare dashboard so that we can redirect all website requests that comes as http to https.

In your cloudflare dashboard, click on Page Rules tab on top


  1. Then click on the button that says Create Page Rule

  2. Type in your domain name http://*mern-stack.com/*

  3. Then from the dropdown settings pick Always Use HTTPS

  4. Then click the button Save and Deploy


So far we used one page rule and two more left to use for free.

Now give it a try. Type http://mern-stack.com/ it will redirect to https://mern-stack.com. Awesome!
