Redirect Digital Ocean IP address to domain name

  1. Login to digital ocean droplet via ssh

  2. Time to configure nginx

  3. cd /etc/nginx/sites-available

  4. sudo vim default

  5. Add additional server block in  > default (put at the end of file) > see below

server {
    listen 80;
    server_name 123.456.789.999;

    return 301 $scheme://domain.com$request_uri;
}


  1. Restart nginx once you have made the above change

  2. sudo systemctl restart nginx

Done! Now anyone who tries to access your site using IP address directly will get auto redirected to domain name!
