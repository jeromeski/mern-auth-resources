Update google login domain and callback url's

Once SSL is active...update google and facebook login domain and callback url's with actual domain name with SSL. So that our google and facebook login will work perfectly in production.

  1. visit console.cloud.google.com

  2. Click on Domain verification > Add A Domain

  1. Popup will appear > verify domain > take me there enter domain name

  2. You will get redirected to verification page


Click add property and choose a verification method



Copy the html tag and paste in your react app /public/index.html



  1. Now push the changes to github > then pull from github to digital ocean

  2. Inside client in local computer

  3. git add .

  4. git commit -m "added meta"

  5. git push

  6. In digital ocean

  7. cd mern-auth-client

  8. sudo git pull

  9. sudo npm run build

  10. pm2 start npm -- start

Now go back to google Webmaster Central and click Verify. It's verified.

P.S. In some cases, you might have to try in a different browser or in private window or try after a while.

  1. Back to console.cloud.google.com

  2. In sidebar, click on OAuth Consent Screen

  3. Enter domain name https://mern-stack.com and hit enter

  4. Then enter same url https://mern-stack.com in home page

  5. Do not enter any other info (unless you have those privacy etc links in your web app)

  6. Click save


  1. Now click in sidebar > Credentials

  2. Click on the link Web Client 1

  3. Authorized JavaScript origins

  4. Remove localhost and Add https://mern-stack.com and hit enter

  5. Under Authorized redirect URIs

  6. Remove localhost and Add https://mern-stack.com and hit enter

  7. Click save

Now open a new browser tab / try private window or clear browser history because it can take some time...
