Redirect all www to non-www

Currently if anyone type www.mern-stack.com they don't see the page. Lets setup a rule in digital ocean so that any request containing www will be auto redirected to non-www format

  1. So it will redirect from https://www.mern-stack.com/ will redirect to https://mern-stack.com/

  2. Click the button that says Create Page Rule

  3. Type in your domain name https://www.mern-stack.com/*

  4. Then from the dropdown settings pick Forwarding URL

  5. Change the Select Status Code to 301 â€” Permanent Redirect

  6. Then type https://mern-stack.com/$1

So far we have create two page rules. With free account you can create up to 3 page rules for free.


P.S. I had issues with this. Well maybe it wasn't issue, all I had to do was wait 24/48 hours (I actually came after 2 days and check to see it was working).
